Описание решения:
1. GeoData - структуры данных, методы поиска и работа с файлом гео-данных.
2. DataLoadTests - тестовое консольное приложение для профилирования/тестирования GeoData.
3. GeoSearch - проект web-приложения для поиска местоположения
    /index.html - пользовательская страница для поиска
    /test.html - страница для тестирования API
